# PROJETApp
J'ai implémenté une architecture hybride I3D et CNN-LSTM pour détecter la violence en vidéo. J'ai utilisé le Flux Optique pour capturer la dynamique temporelle et intégré des CAMs (Class Activation Maps) pour localiser et expliquer les décisions du modèle, assurant ainsi l'interprétabilité du système.
